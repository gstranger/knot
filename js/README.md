# knot-cad

TypeScript/React bindings for the [Knot](https://github.com/gstranger/knot)
NURBS CAD kernel.

Knot is a BREP modeling kernel written in Rust, compiled to WebAssembly. This package wraps the raw WASM bindings with an ergonomic TypeScript API and optional React Three Fiber components for 3D rendering.

## Install

```bash
npm install knot-cad
# or
pnpm add knot-cad
```

React Three Fiber integration requires peer dependencies:

```bash
npm install react three @react-three/fiber
```

## Quick start

```ts
import { createKnot } from 'knot-cad'

const knot = await createKnot()

// Sketch → extrude → boolean → export
const profile = knot.profile([[0,0], [2,0], [2,2], [0,2]])
const block = profile.extrude({ distance: 1 })
const hole = knot.cylinder({ radius: 0.4, height: 2 }).translate(1, 1, -0.5)
const result = block.subtract(hole)

// Export for 3D printing
const stl = result.toSTL()
downloadBlob(stl, 'part.stl')

// Clean up WASM memory
profile.free()
block.free()
hole.free()
result.free()
```

## Architecture

```
knot-cad               (this package — TypeScript wrapper)
  └── knot-wasm         (wasm-pack output — raw WASM bindings + .d.ts)
        └── knot_bg.wasm  (compiled Rust kernel)
```

`knot-wasm` is generated by `wasm-pack build` from the Rust workspace and linked as a pnpm workspace dependency. You never import it directly — `knot-cad` handles WASM loading and provides the public API.

### Two entry points

| Import path | What you get |
|---|---|
| `knot-cad` | Core API: `createKnot`, `Brep`, types |
| `knot-cad/react` | React hooks + R3F component: `useKnot`, `useBrep`, `KnotMesh` |

The React entry point is optional. If you're not using React, only the core entry point is needed and there are no React/Three.js dependencies.

## API

### `createKnot(wasmPath?): Promise<Knot>`

Initialize the WASM kernel. Returns the `Knot` modeling interface. Safe to call multiple times — the kernel loads once and subsequent calls return the same instance.

```ts
const knot = await createKnot()
// or with an explicit .wasm path:
const knot = await createKnot('/assets/knot_bg.wasm')
```

### `Knot` interface

Factory methods that create `Brep` solids:

```ts
// Primitives
knot.box(sx, sy, sz)                         // box centered at origin
knot.sphere({ radius, center?, segments?, rings? })
knot.cylinder({ radius, height, center?, sides? })

// Sketch profiles (for extrude/revolve)
knot.profile([[0,0], [1,0], [1,1], [0,1]])   // 2D polygon (z=0 plane)
knot.profile([{x:0,y:0,z:0}, ...])           // 3D polygon

// Booleans
knot.union(a, b)                              // boolean union
knot.intersection(a, b)                       // boolean intersection
knot.subtraction(a, b)                        // boolean subtraction

// STEP I/O
knot.importSTEP(stepString)                   // parse a STEP file
knot.exportSTEP(brep)                         // write a STEP file
```

### `Brep` class

Wraps an opaque kernel solid. All operations return new `Brep` instances (the kernel is immutable).

```ts
// Transforms
const moved = brep.translate(dx, dy, dz)
const turned = brep.rotate({ x: 0, y: 0, z: 1 }, Math.PI / 4)
const bigger = brep.scale(2)                  // uniform
const stretched = brep.scale(1, 2, 1)         // non-uniform (planar/NURBS only)

// Feature operations (on profile Breps)
const solid = profile.extrude({ distance: 5 })
const solid2 = profile.extrude({ direction: { x: 1, y: 0, z: 1 }, distance: 3 })
const lathe = profile.revolve()               // full 360 around Y axis
const lathe2 = profile.revolve({
  axisOrigin: { x: 0, y: 0, z: 0 },
  axisDirection: { x: 0, y: 1, z: 0 },
  angle: Math.PI,                             // half revolution
})

// Fillet and chamfer (edges identified by endpoint coordinates)
const rounded = brep.fillet([
  { start: { x: -1, y: -1, z: 1 }, end: { x: 1, y: -1, z: 1 } },
], 0.2)
const beveled = brep.chamfer([...edges], 0.1)

// Booleans (chainable)
const result = a.subtract(b)
const combined = a.union(b).intersect(c)

// Tessellation
const mesh = brep.tessellate()
// mesh.positions: Float32Array  (flat xyz)
// mesh.normals:   Float32Array  (flat xyz)
// mesh.indices:   Uint32Array   (triangle indices)

// Export
brep.toSTL()   // → Uint8Array (binary STL)
brep.toGLB()   // → Uint8Array (binary glTF 2.0)
brep.toSTEP()  // → string     (STEP AP203)

// Properties
brep.faceCount // number of faces

// Memory management
brep.free()    // release WASM memory
```

**Memory:** Each `Brep` holds a WASM-side allocation. Call `.free()` when done, or use `using` (TC39 Explicit Resource Management):

```ts
using part = knot.box(1, 1, 1)
// automatically freed when `part` goes out of scope
```

### `MeshData`

Returned by `brep.tessellate()`. Contains GPU-ready typed arrays:

```ts
interface MeshData {
  positions: Float32Array   // [x0,y0,z0, x1,y1,z1, ...]
  normals: Float32Array     // [nx0,ny0,nz0, ...]
  indices: Uint32Array      // triangle index triplets
  vertexCount: number
  triangleCount: number
}
```

Positions and normals are `Float32Array` (converted from the kernel's f64) for direct use with WebGL and Three.js `BufferAttribute`.

## React Three Fiber

### `useKnot(wasmPath?): UseKnotResult`

Hook that loads the WASM kernel. Returns `{ knot, loading, error }` so you can handle all three states.

```tsx
function App() {
  const { knot, loading, error } = useKnot()

  if (error) return <div>Failed to load: {error.message}</div>
  if (loading) return <div>Loading kernel...</div>

  // knot is ready
}
```

Safe to call from multiple components — the WASM module loads once.

### `useBrep(knot, factory, deps): Brep | null`

Hook that creates a `Brep` from a factory function. The factory receives the kernel instance, so you don't need to close over it or add it to deps manually. Automatically frees the previous `Brep` when dependencies change and on unmount.

Returns `null` while `knot` is null (still loading).

```tsx
function Part({ offset }: { offset: number }) {
  const { knot } = useKnot()

  const brep = useBrep(knot, (k) => {
    const a = k.box(2, 2, 2)
    const b = k.cylinder({ radius: 0.8, height: 3 }).translate(offset, 0, 0)
    const result = a.subtract(b)
    a.free()
    b.free()
    return result
  }, [offset])

  return <KnotMesh brep={brep} />
}
```

The factory re-runs whenever `deps` change (like `useMemo`). Intermediate Breps created inside the factory should be freed manually — only the returned Brep is managed by the hook.

### `<KnotMesh brep={...} />`

React Three Fiber component that renders a `Brep` as a `<mesh>` with `BufferGeometry`. Properly disposes GPU resources (calls `geometry.dispose()`) when the brep changes or the component unmounts.

```tsx
import { Canvas } from '@react-three/fiber'
import { OrbitControls } from '@react-three/drei'
import { useKnot, useBrep, KnotMesh } from 'knot-cad/react'

function Scene() {
  const { knot } = useKnot()

  const brep = useBrep(knot, (k) => k.box(2, 2, 2), [])

  return (
    <Canvas camera={{ position: [5, 5, 5] }}>
      <ambientLight intensity={0.4} />
      <directionalLight position={[10, 10, 5]} />
      <KnotMesh brep={brep} color="#4488cc" />
      <OrbitControls />
    </Canvas>
  )
}
```

Props:

| Prop | Type | Default | Description |
|---|---|---|---|
| `brep` | `Brep \| null` | required | The solid to render. Renders nothing if null. |
| `color` | `ColorRepresentation` | `'#4488cc'` | Material color. |
| `tessellateOptions` | `TessellateOptions` | default | Control mesh density. |
| `...rest` | mesh props | | Forwarded to the underlying `<mesh>`. |

## Exporting files

```ts
// Binary STL (3D printing)
const stl = brep.toSTL()
download(stl, 'model.stl', 'model/stl')

// Binary glTF (3D viewers, game engines)
const glb = brep.toGLB()
download(glb, 'model.glb', 'model/gltf-binary')

// STEP (CAD interchange)
const step = brep.toSTEP()
download(new TextEncoder().encode(step), 'model.step', 'text/plain')

function download(data: Uint8Array, filename: string, mime: string) {
  const a = document.createElement('a')
  a.href = URL.createObjectURL(new Blob([data], { type: mime }))
  a.download = filename
  a.click()
  URL.revokeObjectURL(a.href)
}
```

## Building from source

From the repository root:

```bash
# Full build: Rust → WASM → TypeScript
pnpm run build

# Or step by step:
pnpm run build:wasm   # wasm-pack build + fix package name
pnpm run build:js     # tsup bundles js/src → js/dist
```

Requires: Rust toolchain with `wasm32-unknown-unknown` target, `wasm-pack`, Node.js, pnpm.

### Workspace layout

```
knot/
├── crates/            Rust workspace (8 crates)
├── pkg/               wasm-pack output (generated, "knot-wasm")
├── js/                this package ("knot-cad")
│   ├── src/
│   │   ├── kernel.ts        WASM init + Brep wrapper
│   │   ├── index.ts         core barrel export
│   │   └── react/
│   │       ├── use-knot.ts  WASM loader hook
│   │       ├── use-brep.ts  managed Brep lifecycle hook
│   │       ├── knot-mesh.tsx R3F mesh component
│   │       └── index.ts     react barrel export
│   ├── dist/                built output
│   ├── tsup.config.ts
│   └── package.json
├── package.json       root workspace scripts
└── pnpm-workspace.yaml
```

## Bundler configuration

### Vite

```ts
// vite.config.ts
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import wasm from 'vite-plugin-wasm'

export default defineConfig({
  plugins: [react(), wasm()],
})
```

### Next.js

```js
// next.config.js
module.exports = {
  webpack: (config) => {
    config.experiments = { ...config.experiments, asyncWebAssembly: true }
    return config
  },
}
```

### Without a WASM plugin

Serve the `.wasm` file as a static asset and pass the URL explicitly:

```ts
const knot = await createKnot('/knot_bg.wasm')
```
